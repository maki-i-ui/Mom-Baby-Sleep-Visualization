# 母と子の睡眠記録

© 2026 maki-i

## この作品について

**母と子の睡眠記録**は、妊娠中から産後10ヶ月まで、母親と子どもそれぞれの睡眠を月ごとに可視化したインタラクティブな作品です。

数字や統計を主役にするのではなく、眠りの断片と時間の流れを、空の色やチャートの形として体験してもらうことを意図しています。各月には、その時期の日記テキストが添えられています。

## 見方

1. [作品を見る](monthly-layout/) — メインの閲覧ページへ進みます
2. 冒頭の**読み方**で、チャートの見方（スパイラル・棒グラフ・左右の配置など）を確認できます
3. 各月は**月ラベルと日記** → **チャート**の順に並んでいます。左が母親、右が子どもの睡眠記録です
4. 妊娠中は4週（28日）ごと、産後は出生日から30日ごとに区切っています（暦の月ではありません）
5. 各日の睡眠時間は**朝7:00を起点**に24時間で計算しています（午前7時から翌朝7時までを1日とみなします）。生活リズムに沿った区切りとして、深夜の授乳や早朝の起床を同じ「日」に含める意図があります
6. 画面の広い環境（パソコンやタブレットなど）での閲覧を推奨します

## データについて

- 睡眠の記録は Apple Watch 等をもとに、作者が作品用に加工しています
- 公開データの日付・時刻は作品表現のために加工しており、実際の記録日と一致するとは限りません
- 日記テキストは作者による手書きの記述です
